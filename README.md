
# チュウニズム マップ効率周回用プログラム

## 概要 (Overview)
アーケード音楽ゲーム「チュウニズム (CHUNITHM)」における、日々のプレイスケジュールと課金（追加チケット）の最適化を行うPythonツールです。

目標とするマップ進行度やチームポイント、手持ちの無料チケット枚数、日々の予算上限などの現状を入力することで、「いつ・どのチケットを組み合わせて・どのマップ（キャラ指定の有無）をプレイするのが最も効率的か」を数理最適化（混合整数計画法）を用いて自動算出します。

## 特徴 (Features)
- **高度な最適化アルゴリズム**: 日々の予算内で可能なチケットの組み合わせを計算し、「マップ進行特化」「EXP特化」「TP特化」「チケット温存」「バランス型」の5つの軸から最強の行動パターンを抽出してスケジュールを組みます。
- **ボーナス日の考慮**: 「TP2倍」「EXP1.5倍」「マス進捗+2」といった曜日ボーナスや月間イベント日を考慮し、「通常日は節約し、ボーナス日にリソースを集中させる」といった賢い立ち回りを提案します。
- **現実的な制約への対応**: 「今日の課金は1000円まで」「今日は3回しかプレイできない」といったリアルな制約を遵守した上で、目標達成のための最短ルートを導き出します。
- **高速な計算処理**: 動的計画法（DP）とパレート・フロント（劣解の排除）を組み合わせることで、組み合わせ爆発を防ぎ、数ヶ月分のスケジュールでも数秒で計算が完了します。

## 必須要件 (Requirements)
- Python 3.x
- `pulp` (数理最適化ライブラリ・CBCソルバーを利用)

## インストール (Installation)
1. Python環境が構築されていることを確認します。
2. 以下のコマンドを実行し、最適化ライブラリの `pulp` をインストールします。
   ```bash
   pip install pulp
   ```

## 使い方 (Usage)
1. ターミナル（コマンドプロンプト等）でスクリプトを実行します。
   ```bash
   python chunithm_map_2.py
   ```
2. コンソールの案内に従って、現在の進捗や目標数値を対話形式で入力します。
   主な入力項目は以下の通りです。
   - 今日の日付
   - プレイ曲数設定 (3曲 or 4曲)
   - 今月・来月の目標TPまでの残りポイント
   - キャラ指定あり / なしマップの残り必要マス数
   - 手持ちのEXP6倍 / 9倍チケットの残り枚数
   - 今日の予算上限 / プレイ回数上限
   - 各種ボーナスの適用曜日や日付
   *(※未入力のままEnterを押すと、カッコ内のデフォルト値が適用されます)*

3. 計算が完了すると、コンソールに以下の情報が出力されます。
   - **今日の推奨アクション**: 推奨マップ（キャラ指定の有無）と、具体的なプレイ内容・消費金額。
   - **明日以降のスケジュール見通し**: ボーナス日や追加課金が推奨される日のピックアップ。
   - **最終到達見込みリザルト**: 最終的な総課金額、獲得マップ数、獲得EXP、TPのシミュレーション結果。

## 内部仕様について
本ツールは最適化計算において、以下のステップを踏んでいます。
1. **組み合わせの生成**: 指定された予算上限内で、「3倍チケット」「2倍＋1倍」「通常プレイ複数回」など、あらゆるチケットの部分和（ナップサック問題的アプローチ）を算出します。
2. **精鋭の抽出**: 生成された数百のパターンから、各コスト帯につき「マス特化・EXP特化・TP特化・節約特化・バランス」のTop5ずつ（最大25パターン）だけを抽出して重複を排除します。
3. **ソルバーによる割り当て**: 抽出された少数の「神パターン」を変数としてPuLPに渡し、残りの日数とノルマに対して最も総合スコア（eval_score）が高くなるようにスケジュールを組み立てます。
